# 사용자 페르소나 10종 — 이별 회복 앱 Reason

> **목적**: 같은 질문도 사용자의 이별 맥락·성격에 따라 *치유*가 되거나 *상처*가 된다.
> 예: 가스라이팅 피해자에게 "상대 장점을 떠올려보자"는 질문은 자기 판단을 다시 무너뜨리는 트리거다.
> 이 문서는 앱이 분기해야 할 10개 페르소나 + 페르소나별 권장/금기 액션 + 온보딩 식별 질문을 정리한다.

> **주의**: 페르소나는 *고정 라벨*이 아니라 *현재 상태에 대한 임시 가설*이다. D+N이 진행되며 바뀔 수 있고, 한 사람이 2~3개 페르소나의 특성을 동시에 가질 수 있다. 앱은 *주(主) 페르소나 1개 + 부(副) 1개*까지 추적한다.

> **분류 축**: ① 애착유형(회피/불안/안정) × ② 이별주체(찬 사람/차인 사람/합의) × ③ 관계 건강도(건강/유해)

---

## P1. 가스라이팅 피해형 (Gaslighting Victim)


| 항목           | 내용                                                                          |
| ------------ | --------------------------------------------------------------------------- |
| **이별 맥락**    | 정서적 학대·통제·반복적 현실 부정을 겪은 관계. 본인이 끝냈거나 상대가 떠난 후에도 "내가 잘못한 건 아닐까"가 머릿속을 떠나지 않음 |
| **핵심 사고 패턴** | 자기 판단 불신("내 기억이 맞나?"), 과도한 자기 검열, 상대 입장 과대 변호                               |
| **위험 신호**    | "그래도 그 사람은…" 패턴 반복 / 본인 단점만 길게 적고 상대 단점은 비워둠 / 직접적 학대 묘사를 "오해였을 수도"로 정정함    |


### ✅ 권장 액션

- **현실 검증(reality testing) 우선**: 시간순 사건 기록 — "그날 무슨 일이 있었어" (해석 X, 사실만)
- 상대 행동을 *제3자 관점*에서 묘사하게 유도 ("친구가 같은 일 겪었다면 뭐라고 했을까?")
- 학대 패턴 심리교육 콘텐츠 우선 노출(가스라이팅의 4단계 등)
- 위기 핫라인을 평소보다 일찍, 명시적으로 노출

### ❌ 금기 액션

- **"상대의 장점을 떠올려보자" 류의 균형 질문** — 이미 과도하게 변호 중인 사용자에게는 자기 판단을 무너뜨리는 재트라우마
- 양가감정 슬라이더(원망↔애정)를 초기에 강요 — 애정 쪽으로 끌려가는 죄책감 자극
- "둘 다의 문제일 수 있어" 식 GPT 응답
- 합의/소통 권유 — 학대 관계에서 소통은 통제 도구로 쓰임

### 🔧 앱 분기

- `/about-me/strengths` 트랙에서 *상대 장점* 항목 비활성화
- 나침반 verdict에서 "너의 잘못이 아니야" 문구 강화
- 졸업 7일 유예 → **3일로 단축 옵션** 제공

## P2. 회피형 애착 (Avoidant Attachment, Dismissive)


| 항목           | 내용                                                             |
| ------------ | -------------------------------------------------------------- |
| **이별 맥락**    | 친밀해질수록 답답함을 느껴 본인이 끝낸 경우가 많음. 또는 상대가 회피 패턴에 지쳐 떠난 경우           |
| **핵심 사고 패턴** | "사실 별거 아니야", "혼자가 편해", 감정을 인지하기 전에 합리화로 덮음                     |
| **위험 신호**    | 일기를 매우 짧게 씀("괜찮음") / D+1~~3에 감정 온도 7~~9 / 갑자기 D+10~14에 폭발적 그리움 |


### ✅ 권장 액션

- **신체 신호 체크인**부터(잠/식욕/멍함) — 감정 직접 묻기 전 우회
- "괜찮다"는 답에 한 단계 더: "괜찮음 안에 0.1%라도 다른 게 섞여 있다면?"
- 짧은 글쓰기 허용(1줄 일기) — 진입 장벽을 낮춤
- D+10 전후 *지연된 슬픔*에 대한 심리교육 선제 제공

### ❌ 금기 액션

- 긴 자유 서술 강요 — 빈 입력란에 부담만 느끼고 이탈
- "감정을 충분히 느껴봐" 식 추상적 권유
- 반복 푸시 알림 — 통제받는다는 감각으로 회피 강화

### 🔧 앱 분기

- 일기 미니 모드(이미 구현됨)를 기본값으로
- 푸시 알림 빈도 -50%
- 나침반 트리거를 D+10 이후로 늦춤

## P3. 불안형 애착 (Anxious Attachment, Preoccupied)


| 항목           | 내용                                                 |
| ------------ | -------------------------------------------------- |
| **이별 맥락**    | 상대의 거리감에 매달리다 끝남. 차였든 합의든 *재회*가 머릿속을 떠나지 않음        |
| **핵심 사고 패턴** | "내가 더 노력했으면", "지금 연락하면…", 분리불안의 신체화(가슴 통증/과호흡)     |
| **위험 신호**    | 새벽 시간 접속 빈번 / "잡고싶어" 방향 반복 선택 / 일기에 상대 이름·연락 시도 흔적 |


### ✅ 권장 액션

- **충동 지연 도구** 강화: "지금 연락하고 싶다" 버튼 → 10분 호흡 + "10분 후에도 같은 마음이면 다시 와"
- 신체 진정 가이드(4-7-8 호흡, 그라운딩 5-4-3-2-1)
- 재회 충동을 *판단하지 않고* 기록 — "오늘 몇 번 연락하고 싶었는지"만 카운트
- 졸업 유예를 *길게* (7일 → 14일 옵션)

### ❌ 금기 액션

- "그 사람의 단점을 떠올려봐" — 일시적으로 효과 있어 보이지만 죄책감 반동으로 더 매달림
- 감정 온도 1~2일 때 "잘 견디고 있어" 식 칭찬 — 감정의 정당성을 박탈한다고 느낌
- 강한 결정 압박("이제 보내자")

### 🔧 앱 분기

- 위기 모달의 새벽 트리거를 00~~04시 → **22~~05시**로 확장
- 추억 정리 트랙(/memory)을 D+21 이후로 지연 노출
- "보내고싶어" 방향 선택 후 24시간 내 연락 시도 감지 → 부드러운 회수 질문

## P4. 갑작스러운 통보 피해형 (Blindsided)


| 항목           | 내용                                             |
| ------------ | ---------------------------------------------- |
| **이별 맥락**    | 일방적 통보·잠수이별·환승이별. *충격*과 *현실 부정*이 주된 정서         |
| **핵심 사고 패턴** | "왜 그랬는지 모르겠어", 답을 찾기 위한 강박적 반추, 자기 가치 손상       |
| **위험 신호**    | 같은 질문 반복 작성 / 상대 SNS 추적 흔적 / "내가 뭘 잘못했지" 자책 일기 |


### ✅ 권장 액션

- **답이 없는 질문에 답을 찾지 않아도 된다는 메시지** 우선 — 종결(closure)은 상대가 주는 게 아니라 본인이 만드는 것
- 충격 단계(D+0~7) 심리교육: "지금 정상적인 반응이야"
- 사실 정리 워크시트: "내가 아는 것" vs "상상한 것" 구분
- 자존감 회복 트랙(Rosenberg 척도 기반) 조기 진입

### ❌ 금기 액션

- "그 사람도 이유가 있었을 거야" — 상대 변호로 들림
- 빠른 졸업·정리 권유 — 충격 단계는 시간이 필요
- "왜 헤어진 거 같아?" 류의 분석 질문 D+14 이전 금지

### 🔧 앱 분기

- D+0~7은 *입력보다 읽기 콘텐츠* 비중 높임
- 나침반 첫 진입을 D+14로 지연
- 위기 모달 자동 발동 임계치 낮춤(2일 연속 → 1일)

## P5. 본인이 끝낸 죄책감형 (Initiator's Guilt)


| 항목           | 내용                                                                 |
| ------------ | ------------------------------------------------------------------ |
| **이별 맥락**    | 본인이 결정해서 끝냈지만, 상대의 고통이 머릿속을 떠나지 않음. *내가 나쁜 사람일까* 반복                |
| **핵심 사고 패턴** | 결정의 정당성 의심, 상대 회상 시 죄책감으로 합리화·미화                                   |
| **위험 신호**    | 상대 장점만 길게 적음(P1과 정반대 동기) / "다시 만나야 하나" 반복 / 결정 직후가 아니라 D+30 전후 흔들림 |


### ✅ 권장 액션

- **결정의 *과정*을 회상**시킴: "왜 끝냈는지 그날의 일기를 다시 보자"
- 죄책감 ≠ 잘못의 증거 — 심리교육 카드
- 상대를 위한 *간접적 자비* 표현 허용("그 사람도 잘 지내길 바라" 같은 한 줄)
- 졸업 후 D+30, D+60 *재방문 회상 의식*

### ❌ 금기 액션

- "상대의 단점을 떠올려봐" — 죄책감 반동으로 더 미화함
- "네 결정이 맞았어" 식의 단정 — 죄책감을 부정당했다고 느낌
- 재회 권유성 질문

### 🔧 앱 분기

- 결정 직후 일기를 *고정 핀*으로 보관 → 흔들릴 때 복기용
- pros-cons 회상 시 *본인이 적은 단점*만 우선 노출
- 위기 모달 트리거에 "재회 충동" 분기 추가

## P6. 반복 재회 사이클형 (On-Off Cycler)


| 항목           | 내용                                                  |
| ------------ | --------------------------------------------------- |
| **이별 맥락**    | 같은 사람과 헤어졌다 다시 만나기를 3회 이상 반복. *결정 피로*가 누적된 상태       |
| **핵심 사고 패턴** | "이번엔 다를 거야" / "그래도 익숙해서 편해" / 패턴 인식 부재              |
| **위험 신호**    | 온보딩에서 "이번이 N번째 이별" 응답 / 졸업 후 7일 내 재가입 / 같은 결정·취소 반복 |


### ✅ 권장 액션

- **패턴 시각화**: 이전 이별-재회 사이클을 타임라인으로 보여줌
- 매 사이클마다 *반복되는 트리거* 식별 워크시트
- 재회 시 충족된 욕구 vs 다시 깨진 부분 구분
- 졸업 유예 14일 + 재진입 시 "지난번엔 어땠어?" 회고 강제

### ❌ 금기 액션

- 단순 "이번엔 끝내자" 응원 — 반복 자체가 패턴이므로 의미 없음
- 매 사이클을 *별개 이별*로 처리(데이터 단절)

### 🔧 앱 분기

- 사용자 프로필에 *사이클 카운터* 표시
- 졸업 후 30일 내 재가입 시 이전 데이터 보존·연결
- 나침반 verdict에 "패턴" 차원 추가

## P7. 첫 이별 충격형 (First Heartbreak)


| 항목           | 내용                                          |
| ------------ | ------------------------------------------- |
| **이별 맥락**    | 첫 진지한 연애의 종결. 비교 대상이 없어 *세계가 무너진* 감각        |
| **핵심 사고 패턴** | "다신 이런 사람 못 만나", "내 인생은 끝났어", 회복 가능성 자체를 의심 |
| **위험 신호**    | 연령대 어림 / 관계 기간 짧지만 감정 강도 매우 높음 / 미래 상상 거부   |


### ✅ 권장 액션

- **회복 가능성 심리교육** 우선: "첫 이별이 가장 강렬하게 느껴지는 *과학적 이유*"
- 시간 경과에 따른 회복 곡선 시각화(다른 사용자 익명 데이터)
- 작은 일상 활성화 미션(밖에 5분 나가기 등)
- 또래 익명 메시지 노출 — *나만 이런 게 아니야* 정상화

### ❌ 금기 액션

- "더 좋은 사람 만날 거야" — 비교 대상이 없는 사용자에겐 와닿지 않음
- 관계 분석 트랙 조기 노출 — 분석할 대조군이 없음
- "성숙한 결정을 해야지" 류 어른스러움 강요

### 🔧 앱 분기

- 온보딩에서 "이번이 처음" 체크 시 콘텐츠 톤 변경(친구 같은 말투)
- 위기 핫라인을 더 잘 보이게(특히 청소년 핫라인 우선)
- 나침반 진입을 D+21로 늦춤

## P8. 권태로 끝난 장기 관계형 (Long-Term Burnout)


| 항목           | 내용                                                            |
| ------------ | ------------------------------------------------------------- |
| **이별 맥락**    | 3년 이상 관계가 *천천히 시들어* 끝남. 강한 사건 없이 합의로 종결된 경우 많음                |
| **핵심 사고 패턴** | "잘못한 건 없는데 왜 슬프지", 정체성의 절반이 사라진 감각, 미래 공백                     |
| **위험 신호**    | 감정 온도가 *완만히 낮은* 상태(3~4) 지속 / 분노·슬픔 같은 강한 정서 부재 / 정체성 질문에 빈 응답 |


### ✅ 권장 액션

- **정체성 재구성 트랙** 우선: "관계 안에서 잃어버린 나" 워크시트
- 새로운 일상 루틴 만들기(혼자만의 의식 설계)
- *작은 슬픔의 정당성* 메시지 — 큰 사건 없어도 슬플 수 있음
- 함께 쌓은 시간을 *무효화하지 않는* 회상 의식

### ❌ 금기 액션

- "왜 헤어졌는지 분석" — 명확한 이유가 없어 자책으로 흐름
- 강한 감정 라벨 강요(분노/배신감) — 해당 없음
- 빠른 새출발 권유

### 🔧 앱 분기

- 일기 감정 라벨에 "공허/멍함/시들음" 추가
- 정체성 트랙(/about-me/identity)을 메인 동선으로
- 함께한 추억을 *부정 없이* 보관하는 옵션(태우지 않고 봉인)

## P9. 헌신 소진형 (Codependent / Over-Giver)


| 항목           | 내용                                                          |
| ------------ | ----------------------------------------------------------- |
| **이별 맥락**    | 본인을 갈아 넣은 관계가 끝남. *상대가 없으면 내가 누군지 모르겠는* 상태                  |
| **핵심 사고 패턴** | 상대의 감정·상태를 본인 것보다 먼저 걱정, 자기 욕구 인식 불가                        |
| **위험 신호**    | 일기에 본인 감정보다 상대 추측이 많음 / "그 사람 잘 지내려나" 반복 / 본인 욕구 질문에 "모르겠어" |


### ✅ 권장 액션

- **자기 인식 회복** 우선: "오늘 *너만의* 작은 욕구 1개" 매일 질문

- 글쓰기 표출(부치지 않을 편지) 기능 제공 — **단, 표출 후 *2차 정서 탐색* 단계와 반드시 결합** (단순 venting은 카타르시스 가설 반증 — Bushman 2002 등). **1일 venting 횟수 상한** (예: 2회) 적용
- 신체 활성화(운동·찬물 세수)로 각성 조절

### ❌ 금기 액션

- "용서가 필요해" — 분노 단계에서 용서 강요는 역효과
- "그 사람도 이유가 있었을 거야" — 가스라이팅처럼 들림

- **단순 venting 무제한 허용** — 분노 반복 표출은 분노 회로를 강화함, 2차 정서 결합 없는 거칠게 모드 금지

### 🔧 앱 분기

  - **거칠게 모드 종료 시 *2차 정서 카드*(슬픔·두려움·상처·수치·무력감) 1화면 강제 노출** — 1개 선택 후에만 저장 완료

# 구현 시 주의사항

1. **페르소나 라벨은 사용자에게 노출하지 않는다** — "당신은 회피형입니다" 식 단정은 자기충족적 예언이 됨.
2. **페르소나는 D+30, D+60에 재추정한다** — 회복 단계에 따라 주 페르소나가 바뀜(예: P10 → P5 → P8 이행 흔함).
3. **모든 분기는 사용자 설정에서 끌 수 있어야 한다** — 자율성 침해 방지.
4. **데이터 윤리**: 페르소나 추정 결과는 본인 외 누구에게도(분석/마케팅 포함) 공유하지 않음을 온보딩 동의서에 명시.
5. **CLAUDE.md 절대 규칙 준수**: 페르소나별 분기 결과로 *판단 문구·비난 문구*를 만들지 않는다. "정답이 아니야" 원칙은 모든 페르소나에서 유지.
