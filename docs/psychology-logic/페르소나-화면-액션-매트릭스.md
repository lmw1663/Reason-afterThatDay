# 페르소나 × 화면 액션 매트릭스 — 분기 빠짐없이 명세

> **목적**: `구현계획.md`의 §2-3-F가 5개 분기점만 다뤄 13개 페르소나의 코드 액션이 누락된 상태였음. 본 문서는 **20개 페르소나 × 11개 화면 = 220 분기 셀**을 빠짐없이 채워 *분류만 되고 행동은 없는* 상태를 차단한다.
>
> **근거 문서**: `페르소나.md` P1~P10 / `페르소나-분류체계.md` P11~P20 / `구현계획.md` §2-3-F
>
> **사용 원칙**:
> - 셀이 "표준"이면 분기 없이 P12 안정형과 동일 동작
> - 셀에 명시된 액션은 *모두* `구현계획.md` Task 2-3-F·2-3-G에 코드 위치를 1:1 매핑
> - 한 사용자가 주+부 페르소나를 가지면 §4 다중 페르소나 우선순위 규칙으로 해소

# 4부 — 다중 페르소나 충돌 해소 규칙

> `usePersonaStore`에서 주(primary) + 부(secondary)가 동시에 분기를 요구할 때 어느 쪽을 따를지 결정하는 규칙. `페르소나-분류체계.md` §5의 텍스트 규칙을 *코드 우선순위 알고리즘*으로 변환.

## 4-1. 우선순위 알고리즘 (`utils/personaResolver.ts`)

```typescript
type ResolvedAction = {
  source: 'primary' | 'secondary' | 'merged' | 'crisis_override';
  action: PersonaAction;
};

const HARMFUL_RELATIONSHIP = ['P01', 'P14', 'P20'] as const;
const COMPLEXITY_PERSONAS = ['P15', 'P16', 'P18'] as const;

export function resolveAction(
  primary: PersonaCode,
  secondary: PersonaCode | null,
  screen: ScreenCode,
): ResolvedAction {
  // R0. 위기 모드는 페르소나 무시
  if (isInCrisisLockout()) return { source: 'crisis_override', action: CRISIS_ACTION };

  // R1. P13 사별 단독 처리
  if (primary === 'P13') return { source: 'primary', action: BEREAVEMENT_ACTION };

  // R2. 유해관계 페르소나가 주이면 무조건 우선 (균형 질문 모두 비활성화)
  if (HARMFUL_RELATIONSHIP.includes(primary)) {
    return { source: 'primary', action: getActionFor(primary, screen) };
  }

  // R3. 주가 P12(안정형)이면 부를 실질로 취급
  if (primary === 'P12' && secondary) {
    return { source: 'secondary', action: getActionFor(secondary, screen) };
  }

  // R4. 주+부 둘 다 외부복잡도 페르소나 (P15/P16/P18) → 복잡도 트랙 분리
  if (COMPLEXITY_PERSONAS.includes(primary) && secondary && COMPLEXITY_PERSONAS.includes(secondary)) {
    return { source: 'merged', action: SPLIT_COMPLEXITY_ACTION };
  }

  // R5. 일반 케이스: 주의 권장/금기 우선 적용 + 부의 *금기*만 추가 적용
  const primaryAction = getActionFor(primary, screen);
  const secondaryGuards = getGuardsFor(secondary, screen);  // 금기만
  return { source: 'merged', action: { ...primaryAction, additionalGuards: secondaryGuards } };
}
```

## 4-3. R5 ("부의 금기만 추가") 적용 규칙

부 페르소나의 ❌금기 액션은 무조건 적용. 단 ✅권장 액션은 *주와 충돌 시 무시*. 예:

- 주 P05(죄책감) + 부 P01(가스라이팅) → P01 권장(현실검증)은 적용 안 함 / 단 P01 금기("상대 단점 회상")는 적용 → 결과: 본인 단점만 노출, 상대 단점 비활성화
