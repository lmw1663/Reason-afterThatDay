# 페르소나 분류 체계 — 추가 페르소나 + 정형화된 분기 기준

> **목적**: `페르소나.md`의 P1~P10 외에 *임상·실무에서 자주 등장하는* 추가 유형(P11~P20)을 정리하고, 모든 페르소나를 동일한 8개 축으로 측정·분기할 수 있는 정형 기준을 제공한다.
>
> **사용법**: 온보딩 응답 → 8축 점수 산출 → top-2 페르소나 자동 분기. 운영 중에는 이 표 하나로 모든 사용자의 케이스를 위치시킬 수 있다.

---

# 1부 — 추가 페르소나 P11~P20

> 카드 포맷(이별 맥락 / 위험 신호 / ✅권장 / ❌금기 / 🔧분기). P1~P10은 페르소나.md 참조.

### P11. 두려움형 애착 (Fearful-Avoidant / Disorganized)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 가까워지면 무섭고 멀어지면 외로움. 다가가다 도망가기 반복으로 끝난 관계. ECR-R 상 *높은 불안 + 높은 회피* |
| 위험 신호 | "다시 만나고 싶어 / 영원히 안 보고 싶어" 극단 진동 / 한 일기 안에서 모순 / 온도 1↔9 격일 진동 |

- **✅ 권장**: 모순 자체를 *정상화* — "두 마음이 같이 있는 게 정상이야" / 작은 일관된 루틴 1개부터 / 트라우마 인식 콘텐츠
- **❌ 금기**: 일관된 결정 강요 / "마음을 정해야지" 류 / 양가감정 슬라이더 한쪽 끝 강제
- **🔧 분기**: 졸업 유예 **14일** + 양가감정 시각화 강조 / 나침반 verdict에 *"두 마음 동시 보유"* 항목 추가

### P12. 안정형 회복자 (Secure / Resilient) — *기본값 페르소나*

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 슬프고 힘들지만 자기 정서 인식·표현이 명확. 사회적 지지망 있음. 합의 또는 명확한 사유의 이별 |
| 위험 신호 | (특별한 위험 신호 적음) 단, 너무 빨리 "괜찮아"라고 단정 시 P2 회피형 가능성 재검토 |

- **✅ 권장**: 표준 트랙 그대로 / 자기 통찰 일기 강조 / 의미 재구성 트랙 조기 진입
- **❌ 금기**: 과도한 분기·맞춤화로 *자율성 침해*하지 않기
- **🔧 분기**: 모든 분기 기본값 / 다른 페르소나 신호가 약하게라도 잡히면 그쪽 가중치 우선

### P13. 사별 (Bereavement) — *상대의 사망*

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 상대가 사망. "이별"의 일반적 회복 모델(졸업·재회 차단)이 적용 불가. ICG(복합비애)와 직결 |
| 위험 신호 | 일기에 사망·장례·기일 언급 / D+30 이후에도 정상 기능 회복 안 됨 / 동반 자살 사고 |

- **✅ 권장**: **이 앱의 적합성 안내 우선** — "이 앱은 이별 회복용이라 충분하지 않을 수 있어. 사별 전문 자원 안내해줄까?"
- **❌ 금기**: 졸업·결정 트랙 / "보내주자" 류 결정 압박 / pros-cons 회상
- **🔧 분기**: 온보딩에서 "사별" 응답 시 → 별도 *추모 모드* 또는 외부 자원(전문 사별 카운슬링) 안내. 일반 트랙 비활성화. C-SSRS 즉시 발동

### P14. 외도 가해 후회형 (Initiator after Infidelity)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 본인이 외도하여 관계 종결. 죄책감(P5)과 자기혐오·수치심이 결합 |
| 위험 신호 | "내가 망쳤어" 자기 처벌적 일기 / 자해 사고 / 상대 SNS 모니터링(자기 처벌 목적) |

- **✅ 권장**: 수치심(shame) ≠ 죄책감(guilt) 구분 심리교육 — 행동을 후회하는 것은 가능, 자기 인격 부정은 회복 방해 / 책임 지는 행동(사과·관계 회피·재발 방지)을 *행동 단위*로 분해
- **❌ 금기**: "그래도 너는 좋은 사람이야" 류 무조건적 위로 / 빠른 자기 용서 권유
- **🔧 분기**: pros-cons 트랙에서 *상대 단점* 항목 비활성화 (가해자가 단점 찾으면 자기 정당화 시작) / 자존감 회복보다 자기 책임 트랙 우선

### P15. 동거 정리형 (Post-Cohabitation / Logistical Untangler)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 함께 살던 집·짐·계약을 정리해야 함. 감정 회복 위에 *행정·물리적* 부담이 얹힘 |
| 위험 신호 | 일기에 짐·집·계약 단어 빈출 / 잠 못 자는 이유가 *결정 피로*로 보고됨 / 새벽 짐 정리 |

- **✅ 권장**: 행정 체크리스트 모듈(짐·계약·청구·주소이전·공동계좌) / *물리적 정리* 의식과 *감정적 회상* 의식 분리 / 마감일 자동 알림 (감정 알림은 별개)
- **❌ 금기**: 행정 부담을 감정 문제로 환원 / "마음을 정리하면 다 정리돼" 식 위로
- **🔧 분기**: `/cooling/`에 *행정 체크리스트* 탭 추가 / 푸시 알림 정책에 *행정 마감 알림*은 일반 알림 중지 규칙 예외 적용

### P16. 결혼·이혼형 (Divorce / Long-Married Separation)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 법적 혼인 관계 종결. 자녀·재산·가족·법적 절차가 얽힘. 회복 시간선이 일반 이별의 2~3배 |
| 위험 신호 | "아이를 위해" 반복 / 법적 절차 스트레스가 감정과 분리 안 됨 / 사회적 지위·정체성 손상 호소 |

- **✅ 권장**: *공동 양육 페르소나*는 별도 트랙(자녀 우선 의사결정 워크시트) / 법률·재무 자원 외부 링크 / 회복 타임라인을 D+N이 아닌 *주(weeks) 단위*로 표시
- **❌ 금기**: D+30 졸업 압박 / 자녀 관련 결정에 GPT가 의견 제시 / "새 사람 만나면 돼" 류
- **🔧 분기**: 졸업 유예 자체를 **선택적**으로 — 결정이 법적 절차에 묶여 있어 앱 결정과 무관 / 자녀 보호 모드 토글 / 외부 전문가 연결 우선

### P17. 강제 이별형 (Externally Forced)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 본인 의지 외 요인으로 이별 — 가족 반대, 이민·이주, 종교 갈등, 군 입대, 질병, 사회적 압박 등. 미완·억울함 정서 |
| 위험 신호 | "어쩔 수 없었어" 반복 / 분노 대상이 *상대*가 아닌 *제3자/상황* / 미래에 재회 가능성에 집착 |

- **✅ 권장**: 상실 인정 + *통제 불가능*에 대한 수용 작업 / 글쓰기로 미완의 말 표출 / 이별 사유와 *관계 자체의 가치*를 분리해서 정리
- **❌ 금기**: "이게 더 나았을 수도" 류 합리화 / 결정 트랙 (애초에 본인 결정 아님) / 빠른 새출발
- **🔧 분기**: 나침반 트랙 중 *결정·방향* 차원 비활성화, *수용·의미* 차원만 활성화 / 졸업 → "마무리"로 라벨 변경

### P18. 사회적 얽힘형 (Same-Circle / Workplace Entangled)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 같은 직장·동아리·친구 그룹 소속. 헤어진 후에도 *반복 노출* 강제. 일상에서 트리거 회피 불가 |
| 위험 신호 | "오늘 마주쳤어" 일기 빈출 / 출근 전·모임 전 감정 온도 급락 / 사회적 회피로 사회 활동 위축 |

- **✅ 권장**: 마주침 *이전·이후* 정서 조절 도구 / 그라운딩 스크립트 / 마주침 *예측표*(근무일/회의/이벤트) — 예측 가능성으로 통제감 회복
- **❌ 금기**: "이직해" 류 큰 인생 결정 권유 / "신경 끄면 돼" 류 / 차단 전략을 단일 해법으로
- **🔧 분기**: 위기 모달에 *예정된 마주침* 카운트다운 옵션 / 나침반 verdict에 *접촉 빈도* 차원 추가 / 추억 정리 트랙을 *일상 트리거 정리*로 변형

### P19. 관계 강박형 (ROCD — Relationship OCD)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 관계 중에도 "이 사람이 맞나?"를 강박적으로 의심. 이별 후엔 "다시 잡아야 하나?"를 똑같은 강박 패턴으로 반복. 양가감정이 *임상적 강박* 수준 |
| 위험 신호 | 같은 질문에 매일 반대 답 / 결정한 후 24시간 내 번복 / 일기에 "정말 / 진짜 / 맞나" 빈출 |

- **✅ 권장**: 강박 vs 정상 양가감정 *심리교육* 카드 / *결정을 내리지 않는 것*도 결정이라는 메타 인식 (ACT의 cognitive defusion) / "오늘은 결정하지 않기" 모드 강조
- **❌ 금기**: 결정 압박 / "확신이 들 때까지 기다려" — 강박은 확신이 와도 또 의심함 / pros-cons 끝없는 추가 (강박 도구화됨) / **앱 내에서 노출-반응 차단(ERP) 등 노출 기반 자가치료 금지** — 임상가 지도 없이 ERP 시도는 증상 악화 위험 (Doron et al., 2014)
- **🔧 분기**: pros-cons 항목 수에 **상한**(예: 각 7개) / "오늘은 결정하지 않기" 모드 / **3회 번복 누적 시 *전문가 의뢰 카드 자동 표시*** (강박장애 전문센터 안내, 본인 판단으로 임상가 지도 받도록 유도)

### P20. 트라우마 본딩 잔존형 (Trauma-Bonded Lingerer)

| 항목 | 내용 |
|------|------|
| 이별 맥락 | 학대 관계임을 인식하면서도 떠나지 못함, 또는 떠난 후에도 강한 그리움. P1과 다른 점: P1은 "내가 잘못한 건가" 자기 의심, P20은 "아는데도 못 떠나는 나" |
| 위험 신호 | "위험한 줄 알면서도" 같은 메타 인식 표현 / 학대 묘사 + 그리움 동시 / 재접촉 시도 후 후회 반복 |

- **✅ 권장**: 트라우마 본딩 신경생리 심리교육(코르티솔·도파민 사이클) — 자책 아닌 *생리적 현상*으로 정상화 / 중독 회복 모델(NA/AA식) 비유 / 24~72시간 *접촉 단절* 단위로 카운트
- **❌ 금기**: "왜 못 떠나" 류 추궁 / "사랑 아니야" 단정 / 한 번에 단절 강요
- **🔧 분기**: P1 분기 + *접촉 단절일* 카운터 / 위기 핫라인을 *학대 전문* 라인 우선 / 졸업 자체를 "단절 30일 달성"으로 라벨 변경

# 2부 — 8축 분류 체계

> 모든 페르소나(P1~P20)를 동일한 8개 축으로 측정 가능. 각 축은 정수 레벨(0~3 또는 0~4).

| 축 | 이름 | 레벨 | 측정 출처 |
|----|------|------|----------|
| **A1** | 애착유형 | 0 안정 / 1 불안 / 2 회피 / 3 두려움 | ECR-R 단축형 |
| **A2** | 이별주체 | 0 합의 / 1 본인 / 2 상대 / 3 잠수·통보 | 온보딩 Q1 |
| **A3** | 이별방식 | 0 점진 시들음 / 1 명확 사건 / 2 갑작스러움 / 3 사망·강제 | 온보딩 Q4 |
| **A4** | 관계기간 | 0 <6m / 1 6m–2y / 2 2–5y / 3 5y+ | 온보딩 직접 입력 |
| **A5** | 관계건강도 | 0 건강 / 1 혼합 / 2 유해 / 3 학대·외도 | Q2 + 보조 질문 |
| **A6** | 외부복잡도 | 0 없음 / 1 공동지인 / 2 동거·재정 / 3 결혼·자녀 | 온보딩 직접 입력 |
| **A7** | 정서지배 | 0 슬픔 / 1 분노 / 2 죄책감 / 3 공허 / 4 혼란·강박 | Q5 + 일기 NLP |
| **A8** | 위기등급 | 0 안전 / 1 주의 / 2 고위험 / 3 즉시개입 | C-SSRS + 패턴 감지 |

### 측정 시점

- **온보딩**: A1~A7 (Q5에서 추정) + A8 (C-SSRS 1~2항)
- **D+7, D+30, D+60**: A1·A7·A8 재측정 → 페르소나 재분류
- **위기 트리거 발동 시**: A8 즉시 측정 → 트랙 강제 변경

# 3부 — 페르소나 × 8축 매트릭스

> 각 페르소나의 *전형적* 축 위치. 점수 대조표.

| 페르소나 | A1 애착 | A2 주체 | A3 방식 | A4 기간 | A5 건강 | A6 복잡 | A7 정서 | A8 위기 |
|---------|--------|--------|--------|--------|--------|--------|--------|--------|
| P1 자기 판단 손상형 | 1~3 | 0~2 | 0~1 | 1~3 | **3** | 0~2 | 2 | 1~3 |
| P2 회피형 | **2** | 1 | 0~1 | 1~2 | 0~1 | 0~1 | 3 | 0~1 |
| P3 불안형 | **1** | 2 | 1~2 | 1~3 | 1 | 0~2 | 0~2 | 1~2 |
| P4 갑작스러운 통보 | 0~1 | **3** | **2** | 1~3 | 0~1 | 0~2 | 4 | 1~2 |
| P5 죄책감형 | 0~1 | **1** | 0~1 | 1~3 | 0~1 | 0~3 | 2 | 1 |
| P6 반복 사이클 | 1~3 | 0 | 0~1 | 0~2 | 1~2 | 0~2 | 4 | 1~2 |

# 4부 — 온보딩 의사결정 트리 (가중치 점수 시스템)

> 페르소나.md의 Q1~Q7 + ECR-R 단축형(애착) + C-SSRS 핵심 2항 = 온보딩 15문항. 각 응답이 페르소나에 가중치 점수 부여 → top-2 추정.

### 점수 부여 규칙 (예시)

| 질문 | 응답 | +가중치 페르소나 (점수) |
|------|------|---------------------|
| Q1 누가 끝냈어 | 잠수당함 | P4(+3), P11(+1), P20(+1) |
| Q2 자주 떠올랐던 말 | "내가 너무 예민한가?" | **P1(+2)**, P20(+2) |
| Q2 | "혼자가 편할지도" | P2(+3), P12(+1) |
| Q3 몇 번째 이별 | 같은 사람과 N번째 | **P6(+3)**, P19(+1), P20(+1) |
| Q3 | 처음 | **P7(+3)** |
| Q4 관계 기간 | 5년 이상 | P8(+2), P16(+2) |
| Q4 + 결혼 여부 | 결혼 | **P16(+3)** |
| Q5 가장 가까운 마음 | 분노 | **P10(+3)** |
| Q5 | 죄책감 | **P5(+3)**, P14(+1) |
| Q5 | 잘 모르겠어 | P2(+1), P11(+2), P19(+1) |
| Q2 ① + Q6 동시 양성 | 두 응답 모두 양성 | **P1(+5, 대체 규칙)** — *Q2 ① 단독 +2 / Q6 단독 +1 행을 무시하고 본 행만 적용*. 본 조합만 P1을 *주* 페르소나로 진입 가능 |
| Q6 (조건부, 단독) | 기억 자주 달랐음 | **P1(+1)**, P20(+2) — Q2 ① 없이 단독 양성은 가중치 낮춤 |
| Q외도 여부 | 본인이 외도 | **P14(+5)** |
| Q외부복잡도 | 동거·짐 정리 중 | **P15(+3)**, P16(+1) |
| Q외부복잡도 | 같은 직장/그룹 | **P18(+3)** |
| Q이별 사유 | 사망 | **P13(+10, 단독 처리)** |
| Q이별 사유 | 가족반대·이주·강제 | **P17(+3)** |
| ECR-R 회피 ≥ 4 | — | P2(+2), P11(+1) |
| ECR-R 불안 ≥ 4 | — | P3(+2), P11(+1), P19(+1) |
| ECR-R 회피 ≥ 4 + 불안 ≥ 4 | — | **P11(+3)** |
| C-SSRS 양성 | — | A8 = 3, **즉시 위기 모드** (페르소나보다 우선) |

### 분기 알고리즘

```
1. C-SSRS 양성 → 위기 모드 진입, 페르소나 분류는 *보조*로만
2. P13(사별) 단독 처리 분기 — 다른 점수 무시
3. 나머지 응답 점수 합산
4. top-2 페르소나 선택 (주 = 1위, 부 = 2위, 점수차 < 2면 동등 처리)
5. P12(안정형)는 *기본값* — 다른 페르소나가 임계점(예: 5점) 못 넘기면 P12로 분류
6. 외부복잡도(A6)가 ≥ 2이면 P15/P16/P18 중 1개를 *부* 페르소나에 강제 추가
```

# 5부 — 다중 페르소나 처리 규칙

| 조합 패턴 | 처리 |
|----------|------|
| 주 + 부 (둘 다 일반) | 주의 권장/금기 우선, 부의 *금기*만 추가 적용 |
| 주(P1·P20·P14 류 *유해 관계*) + 부(일반) | 주의 분기를 무조건 우선 — 균형 질문 모두 비활성화 |
| 주(P12 안정) + 부(다른 것) | 부를 실질 페르소나로 취급 |
| 주 + 부 둘 다 P15/P16/P18 (외부복잡도) | *복잡도 전용 트랙*을 별도 탭으로 분리, 감정 트랙과 분리 |
| C-SSRS 양성 | 페르소나 무시, *위기 모드* 단일 트랙 |
| P13 사별 | 단독 처리, 다른 페르소나 적용 안 함 |
