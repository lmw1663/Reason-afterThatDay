# AI And Prompt Policy

## GPT 호출 정책
- 클라이언트 직접 호출 금지
- Edge Function 프록시 경유만 허용
- 기본 모델: `gpt-4.1-mini`

## 프롬프트 기본 컨텍스트
- 사용자 이별 D+N
- 최근 3일 감정 온도 추이
- 최근 방향 변화
- 말투 원칙(단정 금지, 가능성 제시, 비난 금지)

## 역할 분리
- 진단/나침반의 핵심 계산은 앱 로직에서 처리
- GPT는 감정 공감/문장화에 집중

## 실패 대응
- GPT 실패/타임아웃 시 템플릿 fallback 응답 제공
- 재시도/건너뛰기 UX 제공
