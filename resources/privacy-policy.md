# 개인정보 처리방침

**시행일**: 2026-05-03
**버전**: v1.0.0

reason(이하 "서비스")은 「개인정보 보호법」(이하 "PIPA")을 준수하며, 이용자의 개인정보를 다음과 같이 처리합니다.

## 1. 수집하는 개인정보 항목

### 1-1. 회원 식별 정보 (필수)
- OAuth 프로바이더(Google·Apple·카카오) 식별자
- 닉네임 또는 이메일(프로바이더 제공 시)

### 1-2. 서비스 이용 정보 (필수)
- 이별 날짜, 관계 기간
- 일기 본문, 감정 온도, 방향 선택
- 질문 응답, 장단점·이유 입력

### 1-3. 민감정보 (PIPA §23, **별도 동의 필요**)
- **심리 척도 응답** — PHQ-9, GAD-7, Rosenberg, ECR-R, RRS, C-SSRS 등 임상 척도 응답
- **위기 응답** — 자해·자살 사고 관련 응답(C-SSRS)
- **페르소나 자동 분류 결과** — 응답을 바탕으로 추정된 정서 상태 가설

### 1-4. 자동 수집 정보
- 디바이스 식별자, 앱 버전, OS 버전
- 푸시 알림 토큰
- 접속 시각·빈도(이상 패턴 감지용)

## 2. 개인정보 처리 목적

| 항목 | 목적 |
|---|---|
| 회원 식별 정보 | 로그인·회원 식별 |
| 서비스 이용 정보 | 일기·분석·회상 기능 제공 |
| 민감정보(심리 척도) | 정서 상태 가설 분기, 회복 추적 |
| 민감정보(위기 응답) | 위기 신호 감지 + 자원 안내 |
| 자동 수집 정보 | 서비스 안정성·이상 패턴 감지 |

## 3. 자동화된 의사결정 (페르소나 분류)

본 서비스는 이용자 응답을 바탕으로 **자동으로 페르소나(정서 상태 가설)를 분류**합니다. PIPA §37의2에 따라 이용자는 다음 권리를 갖습니다:

1. **거부권** — 자동 분류를 거부할 수 있습니다(설정 > 페르소나 분류 끄기).
2. **설명 요구권** — 분류 결과 산출 근거를 요구할 수 있습니다.
3. **재처리 요구권** — 분류 결과 정정·삭제를 요구할 수 있습니다.

페르소나 라벨은 **사용자에게 직접 노출되지 않으며**, 내부 분기에만 사용됩니다.

## 5. 제3자 제공

본 서비스는 이용자의 개인정보를 제3자에게 제공하지 **않습니다**. 단, 다음 경우는 예외입니다:

1. 이용자가 사전에 동의한 경우
2. 법령에 의거하거나 수사기관이 적법한 절차에 따라 요청한 경우
3. **응급 상황** — C-SSRS 양성(자해·자살 위험 즉시 개입 등급) 시 119 등 응급 기관에 위치 정보 등 최소한의 정보가 제공될 수 있습니다(이용자가 사전에 별도 동의한 경우에 한함).

## 7. 이용자의 권리

이용자는 언제든 다음 권리를 행사할 수 있습니다(설정 > 데이터):

- **조회**: 저장된 모든 데이터 다운로드
- **수정**: 입력 데이터 수정·삭제
- **삭제**: 회원 탈퇴 + 모든 데이터 영구 삭제
- **반출**: JSON 형식으로 데이터 export
