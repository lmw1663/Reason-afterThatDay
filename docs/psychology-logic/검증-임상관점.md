# 1. 강점 (유지·강화)

1. **"라벨 비노출" 원칙** (페르소나.md §구현 시 주의사항 1)
   자기충족적 예언 차단 — 임상적으로 정확한 판단. 절대 변경 금지.

2. **무료·검증 척도만 선택**
   BDI-2·BAI(유료)를 제외하고 PHQ-9·GAD-7·C-SSRS·Rosenberg로 한정한 결정은 **합법성·접근성·확장성** 모두에서 옳음.

3. **수치심 vs 죄책감 구분 (P14)**
   Tangney·Brené Brown의 연구와 정확히 일치. 회복 작업의 핵심 분기점을 잡았음.

4. **두 차원(잡고싶다↔보내고싶다 × 원망↔애정) 도입**
   psychology-analysis.md §3의 통찰 — 임상에서 *집착-거리두기 차원*만 보는 것의 한계를 정확히 짚음. 이미 코드에 반영됨(`compass/want` 수평축).

5. **"정답이 아니야" 문구 + 졸업 유예**
   인지 유연성·충동성 차단의 핵심 장치. CBT의 *흑백 사고* 차단과 행동심리학의 cooling-off 원칙 모두 충족.

# 2. 임상적 우려사항 (수정 권고)

## 2-1. **C-SSRS 양성 처리 흐름이 비어 있음** ⚠️ 가장 시급

**현재**: 심리검사-도입가능성.md §2-④는 "강도에 따라 핫라인 자동 노출"만 명시.

**문제**: C-SSRS 양성(자살 사고·계획·시도 의도 양성)은 *핫라인 노출*만으로 부족. 임상 표준은 다음 단계를 요구:

1. **앱 기능 일부 *잠금*** — 졸업·결정 트랙 비활성화 (충동 결정 차단)
2. **즉시 *현재 안전 확인*** — "지금 옆에 있어줄 사람이 있어?" 분기
3. **수단 접근성 질문** (means restriction counseling) — C-SSRS Severity & Lethality 부록 항목
4. **응급 자원 *직접 연결*** — 핫라인 표시가 아니라 *원터치 전화 걸기*
5. **24시간 내 *재확인*** 푸시 (CLAUDE.md 알림 금지 원칙의 *예외* 케이스로 명시 필요)

**권고**: `심리검사-도입가능성.md §7`에 "C-SSRS 양성 처치 프로토콜" 별도 절 신설.

---

## 2-2. **"가스라이팅 피해형(P1)" 진단명-유사 라벨 위험**

**문제**:
- "가스라이팅"은 SNS·대중심리에서 광범위하게 *오용*되는 용어. 정상적 의견 충돌도 가스라이팅으로 라벨링되는 경향.
- DSM/ICD에는 없음. *지속적 패턴 + 권력 비대칭 + 의도적 현실 부정* 3요건이 모두 충족되어야 임상적 의미.
- 앱이 P1로 분류 → "내가 가스라이팅 당했구나" 자기 인식 → *오인된* 가해자 낙인 → 정상 관계의 부당한 종결로 이어질 위험.

**권고**:
1. P1 명칭 변경 검토: "**자기 판단 손상형**" 또는 "**현실 검증 필요형**" — 가해자 라벨이 아닌 *내 상태* 라벨로
2. P1 분기 발동 임계점 강화 — Q2(① 응답) + Q6(있어) **둘 다** 양성일 때만 P1 분기. 단독 양성은 *부* 페르소나로만
3. 외부 의뢰 임계 낮추기 — P1 분기 시 *학대 전문 자원*(한국여성의전화 1366, 한국성폭력상담소) 우선 노출

---

## 2-3. **분노 표출(P10) "거칠게 써도 돼" 모드의 함정**

**문제**: 카타르시스 가설(분노를 표출하면 풀린다)은 *경험적으로 반증된* 가설. Bushman(2002) 등의 연구에서 venting은 분노를 *증폭*시킴(rumination 강화).

**현재**: P10 "글쓰기 표출(부치지 않을 편지)" + "거칠게 써도 돼 모드"

**권고**:
- 표출 자체는 유지하되, **반드시 직후 *2차 정서 탐색* 단계와 결합**
- 단독 venting 세션 종료 금지 — "이 분노 아래에 어떤 마음이 있을까?" 한 화면 강제 추가
- 1일 venting 횟수 *상한* (예: 2회) — rumination 차단

---

## 2-4. **P11(두려움형 애착) 임상 난이도 과소 평가**

**문제**:
- Disorganized/Fearful-avoidant는 임상에서 가장 까다로운 애착 유형. 트라우마 이력 동반 빈도 높음(Liotti, Holmes).
- 양가감정 정상화 + 14일 유예만으로는 부족.
- **Bowlby 분리 항의(separation protest)의 *비조직적* 발현** — 자해·해리·관계 의존 동시 발생 가능.

**권고**:
- P11 분기에 *외부 의뢰 권유 임계* 추가 — D+30 시점 PHQ-9 ≥ 10 또는 RRS 상위 30%일 때 외부 자원 안내
- "두 마음 동시 보유" 정상화 메시지에 *"전문가와 함께 보면 더 명확해질 수 있어"* 부드러운 의뢰 문구 포함

---

## 2-5. **P19(ROCD) 자가 ERP 위험**

**문제**: ROCD(관계 강박)는 *불안장애*의 하위 범주. ERP(노출-반응 차단)는 효과적이지만 **임상가 지도 없이 자가 적용 시 증상 악화** 사례 보고됨(Doron et al., 2014).

**현재**: 페르소나-분류체계.md P19에 "ERP 변형" 권장.

**권고**:
- 앱에서 ERP 직접 제공 금지
- 대신 "강박 vs 정상 의심 구분" 심리교육 + *전문가 의뢰 우선* 분기로 변경
- "결정을 내리지 않는 것도 결정" 메타 인식은 유지(이건 ACT의 cognitive defusion에 가까워 안전)

---

## 2-6. **P13(사별)을 페르소나에 포함시킨 것 자체 재검토**

**문제**: 사별은 *이별*과 신경생리·정서·치료 모델이 다름. Worden 4과제 중 *상실 인정* 단계부터 다르고, ICG/PG-13는 별도 임상 진단(DSM-5-TR 2022 신설).

**현재**: P13으로 포함 + "외부 의뢰" 분기.

**권고**:
- P13을 페르소나 *체계 밖*으로 분리 — "이 앱은 사별 회복용이 아닙니다" 온보딩 첫 화면에서 명시 + 사별 전문 자원으로 이관
- 페르소나 체계는 *생존자가 떠난* 케이스에 한정
- 또는 별도 모듈로 분기(현재 부록의 "별도 모듈 필요 케이스"로 이전)

---

## 2-7. **DSM-5-TR 갱신 미반영 (Prolonged Grief Disorder)**

**문제**: 심리검사-도입가능성.md는 ICG를 사용하지만, **DSM-5-TR(2022)에서 PGD(Prolonged Grief Disorder)가 정식 진단**으로 신설됨. 기준: 성인 *12개월 이상* 지속.

**현재**: 문서는 "6개월 이상"으로 기술.

**권고**: ICG → **PG-13** 또는 **TGI-SR+**로 갱신. 기준 시점도 12개월(성인) / 6개월(소아·청소년)로 정정.

---

## 2-8. **연락 시도 자동 감지(P3)의 프라이버시 문제**

**현재**: P3 §🔧 분기 — *"보내고싶어 방향 선택 후 24시간 내 연락 시도 감지"*

**문제**: 연락 시도를 어떻게 감지하는가? 통화 기록·SMS 접근은 한국 PIPA 위반 우려. SNS 접근도 마찬가지.

**권고**:
- 자동 감지 삭제 → *사용자 자가 보고* 카운터로 변경 ("오늘 연락하려 했어?" 1탭 응답)
- 자가 보고만으로 충분히 패턴 추적 가능

---

## 2-9. **변별 타당도(Discriminant Validity) 미검증**

**문제**: P1과 P20, P3와 P11, P5와 P14 등 *임상적으로 매우 가까운* 페르소나가 점수 가중치만으로 변별 가능한지 검증 안 됨.

**권고**:
- 베타 사용자 50~100명 대상 *재현성 테스트* 필요
- 같은 사용자가 1주일 후 재측정 시 같은 페르소나로 분류되는지(test-retest reliability)
- 임상심리 전문가 평가와 일치도(inter-rater)

---

## 2-10. **온보딩 피로도 — 회피형(P2)·통보 피해형(P4) 이탈 위험**

**문제**: 15문항 + 8축 측정은 회피형(긴 입력 거부)과 충격 단계 사용자에게 부담.

**권고**:
- 온보딩 1단계는 *5문항 이내* — Q1, Q3, Q4, Q5, C-SSRS 1항만
- 나머지는 D+1~7에 *분산 측정*
- "지금 길게 답하고 싶지 않아" 한 번 누르면 모든 추가 측정 *건너뛰기* 옵션 강제 제공

# 3. 누락된 임상 프레임워크

| 프레임 | 적용 가능 페르소나 | 이유 |
|--------|----------------|------|
| **자기연민(Self-Compassion, Neff)** | P1, P5, P14, P20 | 죄책감·수치심 페르소나의 핵심 회복 자원. 한국 문화에서 특히 부족한 자원 |

| **Worden 4과제 (Tasks of Mourning)** | 전체 | psychology-analysis.md에 언급되지만 페르소나 분기에 반영 안 됨 |
| **Continuing Bonds (Klass et al.)** | P8, P16, P17 | 장기 관계의 *완전 단절*이 아닌 *재구성*된 연결 — 추억 정리 트랙 보강 근거 |

# 4. 안전·윤리 보강 필요

1. **프로파일링 동의** — 페르소나 분류는 GDPR Art.22 / 한국 PIPA의 *프로파일링*에 해당. 명시적 옵트인 + 분류 결과 *조회·삭제 권리* 보장 필요
2. **위기 양성 데이터 보존 정책** — C-SSRS 양성 응답은 의료기록에 준함. 보존 기간·접근 권한·삭제 정책 별도 명시
3. **AI 응답의 임상 면책** — GPT 응답이 *치료 대체*가 아님을 모든 응답 하단에 명시
4. **외부 의뢰 임계점 *명문화*** — 어느 검사 점수에서 외부 자원 안내가 *자동* 발동되는지 표로 정리 (현재는 "고려" 수준에 그침)
5. **한국 핫라인 정확성 검증** — 자살예방상담전화 1393, 정신건강위기상담 1577-0199, 청소년 1388, 여성긴급전화 1366, 가정폭력 1366 — 페르소나별 우선순위 매핑 필요

# 5. 문화적 맥락 보강

- **한국적 이별 변수 누락**: 가족 반대·결혼 전제 동거·예식 취소·종교 갈등·취집/취집 압박 — P17(강제 이별)에 부분 반영되지만 *한국 특수 시나리오*는 별도 카드 가치 있음
- **성역할 비대칭 침묵**: 데이트 폭력 발생률·가해 비율의 성별 비대칭(여성 피해 다수) — P1·P20 분기에 성별 변수 명시적 고려 필요
- **LGBTQ+ 페르소나 부재**: outing 위험·가족 분리·법적 보호 부재 등 한국 맥락 특수 변수
- **존댓말 톤** — 위기 모달에서 *반말* 유지(CLAUDE.md 규칙)는 친밀성 강조 의도지만, **성인 사용자 일부에는 거부감**. P12(안정형)·P16(이혼)·고연령 사용자에게는 존댓말 옵션 권장
